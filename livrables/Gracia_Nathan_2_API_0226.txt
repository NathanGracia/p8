ENDPOINTS

GET  /                   Health check + informations sur le modèle chargé
GET  /images             Liste des images de validation disponibles
GET  /images/{id}/rgb    Image RGB originale (PNG)
GET  /images/{id}/gt     Masque ground truth colorisé (PNG)
POST /predict            Upload image → masque de segmentation prédit (PNG)
GET  /predict/{id}       Prédiction par ID d'image (PNG)

════════════════════════════════════════

INFRASTRUCTURE

Framework   : FastAPI + Uvicorn
Container   : Docker
Proxy HTTPS : Caddy
Hébergement : VPS Linux
CI/CD       : GitHub Actions (déploiement automatique sur git push master)
